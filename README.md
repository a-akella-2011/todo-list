## Features

- **Add tasks** – Users can add new tasks to their list.
- **View tasks** – Display all tasks with numbering.
- **Edit tasks** – Update the text of an existing task.
- **Mark tasks as done** – Complete tasks without losing track.
- **Clear all tasks** – Remove all tasks at once, with confirmation.
- **Search tasks** – Find tasks using keywords (case-insensitive).
- **Optional priority system** – Assign High, Medium, or Low priority to tasks.
- **Persistent storage** – Tasks are saved to a file (`tasks.txt`) so they remain after closing the app.
